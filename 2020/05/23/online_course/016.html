element-ui 使用

本博客记录在使用element-ui时遇到的一些问题和使用经验.

  1. 安装配置element-ui

    npm i element-ui -S

  1. 在main.js文件中引入element-ui组件

    1. 完整引入

      import Vue from 'vue';
import ElementUI from 'element-ui';
import 'element-ui/lib/theme-chalk/index.css';
import App from './App.vue';

Vue.use(ElementUI);

  1. 单独引入要使用的模块

    import Vue from 'vue';
import { Button, Select } from 'element-ui';
import App from './App.vue';

Vue.component(Button.name, Button);
Vue.component(Select.name, Select);
/* 或写为
 * Vue.use(Button)
 * Vue.use(Select)
 */

  2. 完整引入将会把所有可以使用的组件全部引入到我们的项目中去,机试这部分功能不使用,也会进行引入

  3. 单独引入就是只引入我们要使用的组件,想要添加新的组件需要重新引入

  4. 默认为完整引入

构建基本的导航栏

  1. 在这里我选用的是Container 布局容器

  2. 以及侧边导航栏

  3. 布局容器常用属性

    • unique-opened 设定在打开侧边栏的时候仅仅打开一个,其余自动关闭
    • collapse 是否对导航栏进行折叠,默认值为 False,不适用折叠功能,修改后侧边导航栏将会折叠起来,只保留图标
    • router 给导航栏的每一项开启路由跳转功能,默认不开启,使用True开启

  4. 导航栏数据基本格式:

    menu_list: [
    {
          'id': 1,
          'authname': '总览',
          'path': null,
          'icon': 'el-icon-pie-chart',
          'children': [
            {
              'id': 11,
              'authname': '总览',
              'path': 'dashboard',
              'children': []
            },{
              'id': 12,
              'authname': '实时监控',
              'path': 'monitoring',
              'children': []
            }
          ]
    },
