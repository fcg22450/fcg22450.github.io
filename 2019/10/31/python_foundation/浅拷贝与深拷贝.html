Python中copy模块用于对象的拷贝操作；该模块只提供了两个主要的方法：copy.copy与copy.deepcopy，分别表示浅拷贝与深拷贝

一、浅拷贝

　　浅拷贝是对于一个对象的顶层拷贝，相当于只拷贝了最浅的一层或者第一层。通俗的理解是：拷贝了引用，并没有拷贝内容

# 例如：
import copy       # 浅拷贝模块
a = [11,22]
b = [33,44]
c = [a,b]
print(id(a))            # 列表a的内存地址
print(id(b))            # 列表b的内存地址
print(id(c))            # 列表c的内存地址
d = copy.copy(c)      # 将列表c浅拷贝给列表d
print(id(d))           # 列表d的内存地址
print(id(d[0])) # 列表d中下标0的内存地址(因为下标0是列表a，所以内存地址与a一样)
print(id(d[1])) # 列表d中下标0的内存地址(因为下标1是列表b，所以内存地址与b一样)
a.append(11)      # 向列表a中添加元素11
print(c)               # 输出添加后列表c的数据结果
print(d)                 # 输出添加后列表d的数据结果(因为d为浅拷贝c的数据，所以列表d与列表c的输出结果相同)
2677221188168
2677221188232
2677221682376
2677251508296
2677221188168
2677221188232

[[11, 22, 11], [33, 44]]    # 第一层为a,b，并非外面的空列表框
[[11, 22, 11], [33, 44]]    # 第一层为a,b，并非外面的空列表框
浅拷贝只拷贝第一层数据，列表[[11,22],[33,44]]中的第一层数据为[11,22],[33,44]，而并非最外层的空[ ]，所以，当列表a中添加元素时，拷贝的里面同样会添加上这个元素

1、浅拷贝的三种情况

　　1、拷贝不可变对象：只是增加一个指向原对象的引用，改变会互相影响。

a = (1, 2, [3, 4])
b = copy.copy(a)
print(b)
(1, 2, [3, 4])

# 改变一方，另一方也改变
b[2].append(5)
print(a)
(1, 2, [3, 4, 5])

　　2、拷贝可变对象(一层结构)：产生新的对象，开辟新的内存空间，改变互不影响。

import copy

a = [1, 2, 3]
b = copy.copy(a)
print(b)
[1, 2, 3]

# 查看两者的内存地址，不同，开辟了新的内存空间
print(id(b))
print(id(a))
1833997595272
1833997595080

print(a is b)
False

# 改变了一方，另一方关我卵事
a = [1, 2, 3] 
b = [1, 2, 3]
b.append(4)
print(a)
[1, 2, 3]

a.append(5)
print(b)
[1, 2, 3, 4]

　　3、拷贝可变对象(多层结构)：产生新的对象，开辟新的内存空间，不改变包含的子对象则互不影响、改变包含的子对象则互相影响。

import copy
a = [1, 2, [3, 4]]
b = copy.copy(a)
print(b)
[1, 2, [3, 4]]

# 查看两者的内存地址，不同，开辟了新的内存空间
print(id(b))
print(id(a))
1833997596488
1833997596424
print(a is b)
False

# 1.没有对包含的子对象进行修改，另一方关我卵事
a = [1, 2, [3, 4]]
b = [1, 2, [3, 4]]
b.append(5)
print(a)
[1, 2, [3, 4]]

a.append(6)
print(b)
[1, 2, [3, 4], 5]

# 2.对包含的子对象进行修改，另一方也随之改变
a = [1, 2, [3, 4]]
b = [1, 2, [3, 4]]
b[2].append(5)
print(a)
[1, 2, [3, 4, 5]]

a[2].append(6)
print(b)
[1, 2, [3, 4, 5, 6]]

二、深拷贝

　　1、深拷贝会完全复制原变量的所有数据（递归性质的拷贝），在内存中生成一套完全一样的内容，我们对这两个变量中的一个进行任意修改都不会影响另一个变量。

　　注意：浅拷贝对不可变类型和可变类型的copy不同：copy.copy对于可变类型，会进行浅拷贝，copy.copy对于不可变类型，不会拷贝，仅仅是指向

# 例如：
import copy
a = [11,22]
b = [33,44]
c = [a,b]
print("c 的内存地址：",id(c))

d = copy.deepcopy(c)
print("a 的内存地址：",id(a))
print("b 的内存地址：",id(b))
print("d 下标0的内存地址：",id(d[0]))
print("d 下标1的内存地址：",id(d[1]))
print("d 的内存地址：",id(d))

c[-1].append(55)
print("c 添加元素后：",c)
print("d 的输出结果：",d)
输出结果为：
c 的内存地址： 1695548046536
a 的内存地址： 1695547548232
b 的内存地址： 1695547548296
d 下标0的内存地址： 1695549425096     # 与a的内存地址不同，因为只拷贝了a的内容
d 下标1的内存地址： 1695549627720    # 与b的内存地址不同，因为只拷贝了b的内容
d 的内存地址： 1695549426312    
c 添加元素后： [[11, 22], [33, 44, 55]]        # 深拷贝里c的内容发生改变
d 的输出结果： [[11, 22], [33, 44]]            # 拷贝后的内容不会受到任何影响
d是深拷贝c的内容，并没有拷贝内存地址，所以，即便内容相同，内存地址也是不相同的，另外，当d拷贝过c里面的内容后，我们再对c进行修改或添加元素，并不会影响d里面的内容。

三、== 和 is运算符的区别

　　1、== 比较的是两个对象的内容是否相等，即内存地址可以不一样，内容一样就可以了。

　　2、is 比较的是两个实例对象是不是完全相同，它们是不是同一个对象，占用的内存地址是否相同。

　　个人理解：如果两个对象is成立，那么==就一定成立；相反，如果两个对象==成立，并不能代表is就一定成立

# 例如1：
a = [11,22]
b = [11,22]
print(id(a))
print(id(b))
if a == b:
    print("相等的")
if a is b:
    print("a 是 b")
# 输出结果为：
3015265903176
3015265903240
相等的        # 因为if a is b条件不成立，所以没有输出结果
# 例如2：
a = 3
b = 3
print(id(a))
print(id(b))
if a is b:
    print("a 是 b")
if a == b:
    print("相等的")
# 输出结果为：
140708210070624
140708210070624
a 是 b        # 内存地址一样，条件成立，输出结果
相等的        # 内容一样，条件成立，输出结果
